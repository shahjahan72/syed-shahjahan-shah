Portfolio Website

This portfolio was designed and developed to showcase responsive, modern UI with subtle animation using GSAP. The goal was to create a fast, accessible experience highlighting projects and services.

Role: Designer & Front-end Developer

Tech: HTML5, CSS3, Vanilla JS, GSAP

Deliverables: Responsive website, project pages, contact form, deployment-ready build.

Challenge

The main challenge was balancing visual polish (glass effect, hero imagery) with performance and accessibility. Images were optimized and lazy-loaded. Focus states and a skip-link were added for keyboard users.

Highlights

  • Responsive grid layout for projects and services.
  • Light / dark mode toggle.
  • Modal preview for quick project inspection.
  • Netlify-ready form integration (Netlify Forms) for easy contact submissions.
