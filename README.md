# Portfolio Website

A next-generation portfolio website built with React, Vite, Framer Motion, and distinct, premium styling.

## Getting Started

1.  **Install Dependencies** (if not already done):
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## Features

- **Framework**: React 18 + Vite
- **Styling**: Custom CSS Variables + Glassmorphism
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Project Structure

- `src/components`: UI Components (Hero, Navbar, etc.)
- `src/styles`: Global styles and variables
- `_legacy_backup`: Backup of your previous site
